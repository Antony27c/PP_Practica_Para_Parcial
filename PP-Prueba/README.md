# PP-Prueba

Aplicación React con rutas que consume dos APIs públicas:

- **PokeAPI** — Información de Pokémon (nombre, imagen, tipo)
- **RestCountries** — Datos de países (nombre, capital, bandera)

## Rutas

| Ruta          | Página     |
| ------------- | ---------- |
| `/`           | Inicio     |
| `/pokemones`  | Pokemones  |
| `/banderas`   | Banderas   |

## Tech

- React + Vite
- React Router DOM
